# 🌱 Greeny - The Ultimate Agriculture & Plant Platform

![Greeny Banner](https://images.unsplash.com/photo-1500382017468-9049fed747ef?q=80&w=2000&auto=format&fit=crop) > **Where Farms Meet Community.** > Greeny is a comprehensive web-based platform designed to empower farmers, local vendors, and plant enthusiasts. [cite_start]It seamlessly integrates an e-commerce marketplace, a detailed plant encyclopedia, and an active community forum into one unified ecosystem[cite: 3, 4].

---

## 🎉 Key Features

### 🛒 1. Agricultural Marketplace
[cite_start]A multi-vendor e-commerce space connecting buyers and sellers of agricultural goods[cite: 16].
* [cite_start]**Product Browsing:** Filter by price, popularity, and top reviews[cite: 20].
* **Vendor Dashboard:** Sellers can manage their products, track sales, and interact with customers.
* [cite_start]**Seamless Checkout:** Fully functional cart system with secure order placement[cite: 22, 23].

### 📖 2. Plant Reference Wiki
[cite_start]A massive database providing scientific and practical care guides for various plants[cite: 25].
* [cite_start]**Detailed Guides:** Information on soil type, sunlight needs, and planting seasons[cite: 29, 31, 32].
* [cite_start]**Disease Management:** Identification of common plant diseases and their treatments[cite: 33, 34].

### 💬 3. Community Forum & Blog
[cite_start]The social heart of Greeny, allowing users to share experiences and find solutions[cite: 35].
* [cite_start]**Interactive Posts:** Upvote/downvote system for community-driven quality control[cite: 39].
* [cite_start]**Visual Problem Solving:** Farmers can upload photos of affected crops to get expert advice[cite: 44, 45].

### 💧 4. Smart Irrigation Calculator
[cite_start]An advanced tool to optimize water usage.
* [cite_start]Inputs: Crop type, land area, soil type, and farm location[cite: 49, 50, 51, 52].
* [cite_start]Outputs: Recommended irrigation amounts and optimal watering times based on average local temperatures[cite: 53, 54, 55].

### 🌓 5. UI/UX Enhancements
* **Dark/Light Mode:** Fully responsive theme toggler with persistent local storage memory.
* **Glassmorphism Design:** Modern, clean, and intuitive navigation.

## 🛠️ Tech Stack

This project is built using a lightweight, framework-free frontend and a robust backend architecture, ensuring high performance and scalability.

**Frontend:**
* HTML5 (Semantic & Accessible)
* CSS3 (Custom variables, Flexbox/Grid, CSS Modules)
* Vanilla JavaScript (DOM manipulation, LocalStorage, Event Handling)

**Backend & Database:**
* **Framework:** .NET (ASP.NET Core)
* **Architecture:** RESTful API
* **Database:** SQL Server / PostgreSQL (Handling relational data between users, products, and posts).
